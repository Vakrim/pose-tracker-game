# Pose Tracker Game

A web application that uses MoveNet from TensorFlow.js to detect and visualize human poses in real-time from your webcam.

## Features

- Real-time pose detection using MoveNet Lightning model
- Full-page webcam display with maintained aspect ratio
- Skeleton visualization overlay on detected poses
- Built with TypeScript and Vite

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and allow webcam access when prompted.

## Build

To build for production:
```bash
npm run build
```

## Technologies

- TypeScript
- Vite
- TensorFlow.js
- @tensorflow-models/pose-detection (MoveNet)
